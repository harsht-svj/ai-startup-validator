# 🚀 AI Startup Validator – Multi-Agent AI Business Validation Platform

AI Startup Validator is a full-stack AI platform that transforms raw startup ideas into structured business validation reports using multiple AI agents. It analyzes ideas from different perspectives, assigns a startup score, identifies strengths and risks, generates improvement suggestions, and creates an execution roadmap.

# 🧠 AI Workflow

The application processes every startup idea using four independent AI agents.

### 👨‍💼 Investor Agent

Evaluates

- Market Opportunity
- Revenue Potential
- Investment Worthiness

---

### 👤 Customer Agent

Evaluates

- User Value
- Pain Points
- Customer Adoption

---

### 🥊 Competitor Agent

Evaluates

- Existing Competitors
- Competitive Advantage
- Market Saturation

---

### 🧠 Synthesizer Agent

Combines all previous analyses and generates

- Final Startup Score
- Summary
- Recommendations
- Risk Analysis
- Execution Roadmap

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/harsht-svj/AI-Startup-Validator.git
```

## Install Client

```bash
cd client
npm install
```

## Install Server

```bash
cd ../server
npm install
```

## Create .env

```env
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret
GROQ_API_KEY=your_groq_api_key
```

## Start Backend

```bash
npm run dev
```

## Start Frontend

```bash
npm run dev
```
